Display user profile image or initials

## Usage
```jsx live
<div style={{ display: 'flex', gap: 16 }}>
  <Avatar size={64} alt="Albert Einstein" src="/images/albert_einstein.webp" />
  <Avatar size={64} alt="Albert Einstein" />
</div>
```

## Without Image
If the `src` prop is not set, the first letter of the `alt` prop will be used as the text inside a colored circle. If the `alt` prop is not set, a "User" Icon will be displayed.

```jsx live
<div style={{ display: 'flex', gap: 8 }}>
  <Avatar size={32} alt="Albert Einstein" />
  <Avatar size={32} alt="Robert Stark" />
  <Avatar size={32} />
</div>
```

## Custom Text/Icon
If the `children` prop is passed, it will be used as the text/element inside the circle.

```jsx live
<div style={{ display: 'flex', gap: 8 }}>
  <Avatar size={32} alt="Robert Stark">XY</Avatar>
  <Avatar size={32} alt="Robert Stark"><Icon name="IconBuildingBank" /></Avatar>
</div>
```

## Colors Array
By default, the colors used for the circle are chosen from the `colorsArray` prop, which has a default value of `["secondary", "primary", "success", "warning", "#FFA53C", "#F700FC", "#3CD0FF", "#CCFF3C"]`. The color chosen depends on the first letter of the `alt` prop.

```jsx live
<div style={{ display: 'flex', gap: 8 }}>
  <Avatar size={32} alt="A" />
  <Avatar size={32} alt="B" />
  <Avatar size={32} alt="C" />
  <Avatar size={32} alt="D" />
  <Avatar size={32} alt="E" />
  <Avatar size={32} alt="F" />
  <Avatar size={32} alt="G" />
  <Avatar size={32} alt="H" />
  <Avatar size={32} alt="I" />
  <Avatar size={32} alt="J" />
  <Avatar size={32} alt="K" />
</div>
```

You can also pass a custom colors array to the `colorsArray` prop.
```jsx live
function CustomColors(props) {
  const colorsArray = ["blueviolet", "cadetblue", "coral", "indianred", "seagreen", "tomato"]

  return (
    <div style={{ display: 'flex', gap: 8 }}>
      <Avatar size={32} alt="A" colorsArray={colorsArray} />
      <Avatar size={32} alt="B" colorsArray={colorsArray} />
      <Avatar size={32} alt="C" colorsArray={colorsArray} />
      <Avatar size={32} alt="D" colorsArray={colorsArray} />
      <Avatar size={32} alt="E" colorsArray={colorsArray} />
      <Avatar size={32} alt="F" colorsArray={colorsArray} />
      <Avatar size={32} alt="G" colorsArray={colorsArray} />
      <Avatar size={32} alt="H" colorsArray={colorsArray} />
      <Avatar size={32} alt="I" colorsArray={colorsArray} />
      <Avatar size={32} alt="J" colorsArray={colorsArray} />
      <Avatar size={32} alt="K" colorsArray={colorsArray} />
    </div>
  )
}
```

## Size
The `size` prop can be used to set the size of the circle.

```jsx live
<div style={{ display: 'flex', gap: 8 }}>
  <Avatar size={32} alt="A" size={128} />
  <Avatar size={32} alt="A" size={64} />
  <Avatar size={32} alt="A" size={32} />
  <Avatar size={32} alt="A" size={24} />
  <Avatar size={32} alt="A" size={16} />
</div>
```
